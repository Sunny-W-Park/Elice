    • 1. Intro.

    안녕하세요, 박선우입니다.
    풀스택 개발자를 준비중이며, 유저의 니즈를 연구하고 새로운 유저 경험을 개발하는 DevOps를 실현하고 싶습니다.

    2. Dev.

    Django와 Django Rest Framework을 활용하여 백엔드 개발 프로젝트를 2회 진행한 경험이 있습니다. 엘리스에서 JS, React 등 프론트엔드 기술 활용도를 높여 개발 스택의 폭을 넓히고 싶습니다.

    3. At Elice.

    훌륭한 동료 개발자분들과 네트워킹하고, 완성도 높은 프로젝트를 만드는 것이 목표입니다.

    특히 DevOps, 애자일 등을 추구하며 민첩하게 제품을 개발하는 것의 중요성에 공감하시는 분들과 협업해보고 싶습니다.

    4. Others.

    로드사이클 라이딩을 즐깁니다. 지난주 시즌온 했는데, 라이딩 좋아하는 분 있으시다면 언제든 연락주세요! 상단에 Strava 프로필 링크 올려두었습니다.
